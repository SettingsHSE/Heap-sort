Пирамидальная сортировка (она же сортировка кучей) –– классический алгоритм который, пожалуй, должен знать любой программист. Старая добрая «пирамидка» примечательна тем, что в независимости от набора данных у неё одна и та же сложность по времени (причём, очень пристойная) – O(n log n). 
Лучших и вырожденных случаев для неё нет. 

Задача - написать программу сортировки одномерного массива целых чисел по неубыванию с помощью алгоритма пирамидальной сортировки. 
Язык — С++ 
Входные данные читаются из текстового файла input.txt 
Выходные данные записываются в текстовый файл output.txt 

Формат ввода
На вход программа принимает 2 строки: 
в первой строке целое число N - количество элементов для сортировки. 
Гарантируется, что 1 <= N <= 300000. 
во второй строке N целых чисел через пробел. 
Обозначим каждое из N входящих чисел как P, тогда гарантируется, что 0 <= P <= 300000. 

Формат вывода
На выход программа должна выводить 1 строку: 
в первой строке M целых чисел через пробел. 
Все числа должны быть по неубыванию, могут быть повторы, и числа должны быть из первоначального набора. 
Предполагается, что 1 <= M <= 300000, и точно соответствует входящему N.
